## Part B -- The polynomial floor (recalled, established 2026-05-29)
Applying the operators to POLYNOMIALS: they occupy ONLY the bottom face of the cube (branch=0),
four cells: holo p(z), anti p(zbar), mixed p(z,zbar) "poly-analytic", constant. A polynomial
NEVER reaches the transcendental top (branch=1, the log) where eml/eml* live -- it has no log,
is entire, single-valued. The log is essential; eml* transcendental is NOT a polynomial and
cannot be built from finitely many polynomial operations.
FINER (poly-analytic order): smallest n with (d/dzbar)^n f = 0.
  z^3 -> 0 (holo); zbar -> 1; zbar^2 -> 2; ... ; log(zbar) -> INFINITE (never annihilated).
This integer order = the LANDAU LEVEL (quantum number) the detector read in the Landau case.
Algebraic anti (finite order, polynomial/Landau ladder) vs transcendental anti (log, infinite
order, eml*) are QUALITATIVELY distinct kinds of anti-holomorphy.

## Unifying principle
T-triviality and the polynomial-floor are the SAME phenomenon: the object's structure caps the
lamp. A q-series caps T to the identity (q absorbs integer shift), exactly as a polynomial caps
the branch-axis to 0 (no log). The lamp can only reveal what the object's structure permits.
A lamp is informative only when the object has the structure it probes:
  - T informative only if the object depends on tau OUTSIDE q (e.g. E2* via 1/Im(tau)) -- next test.
  - branch-axis informative only if the object has a log (transcendental).
  - S informative on any non-trivial modular object.
