## The reframing
All failed candidates (Psi4, chiral SC, QGT, Kerr/Newman-Janis) are SINGLE-complex-field systems.
A single complex observable provides only z and its conjugate zbar; zbar is determined by z, hence
spinor-lock (d) or reality-lock (a). No single-field object can carry independent continuous zbar.
=> the search target is NOT "a system with a complex observable" but "a system with TWO independent
   complex fields whose physical coupling needs the conjugate of one relative to the other".
## Operational criterion (certified on Anthony's machine)
Cross-conjugate coupling f = z * conj(w), with z = field 1, w = field 2 (separately measured):
  d f / d zbar = 0      (no dependence on the conjugate of its OWN field)
  d f / d wbar = z != 0 (anti-holo content carried by an INDEPENDENT field's conjugate)
COLLAPSE CHECK: if the fields are not independent, w = g(z) (e.g. w=z), then f = z*conj(g(z))
  becomes a function of z, zbar alone => f = z*zbar => back to field+conjugate => RE-LOCKED (a/d).
=> INDEPENDENCE is a PHYSICAL condition (statistical independence / distinct DOF), NOT algebraic.
   The cross-conjugate form is necessary; physical independence of the two fields is the deciding,
   non-algebraic requirement.
## Four-step Plateau-B protocol
 1. Identify a system with TWO complex observables z, w measured SEPARATELY.
 2. Verify they are PHYSICALLY INDEPENDENT (w is not a function of z; uncorrelated / distinct DOF).
 3. Form the natural physical object coupling them; test for cross-conjugate coupling z*conj(w).
 4. If present AND independence holds => the anti-holo carrier is a genuine independent measurement
    => escapes locks (a) and (d) => Plateau-B signature.
## Seed of the exhaustion argument (Milo condition 3, single-observable half)
Single complex observable => only {z, zbar} available => zbar determined by z => always locked
(a reality, or d spinor). Therefore the conjecture is PROVABLE for the single-observable case; the
ONLY open door is the two-independent-field case. This halves the exhaustion problem.
