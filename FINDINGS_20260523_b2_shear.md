# FINDINGS — B2 Clunie–Sheil-Small shear calibration (eml / eml-star detector)

**Date:** 2026-05-23
**Status:** [ESTABLISHED] — executed on Anthony's machine AND certified by the
exact SymPy judge (verify_exact.py, Wirtinger derivative d/d(zbar)).
**Nature:** CALIBRATION on a planar harmonic mapping. NOT a discovery.

## 1. What was tested
Clunie–Sheil-Small shear f = h + conj(g). Calibration case (closed form known,
injected on purpose): phi(z)=z, omega(z)=z, so h(z)=-log(1-z),
g(z)=-z-log(1-z).
  D0 control (pure h):   f = h(z)             -> expected HOLOMORPHIC
  D1 shear:              f = h(z)+conj(g(z))  -> expected ANTI-HOLOMORPHIC
Data: 400 pts, |z|<=0.7, area-uniform. Generated by b2_shear_run.py --gen.
Calibration: classes known in advance. Confirms the pipeline SEPARATES holo
from anti and does not call the conjugation-free control anti. Not a discovery.

## 2. Exact commands
  python3 b2_shear_run.py --gen
  python3 b2_shear_light.py --which holo
  JULIA_NUM_THREADS=4 python3 b2_shear_light.py --which shear
  python3 b2_shear_light.py --report
Config: niterations=80, population_size=100, maxsize=18, precision=64,
multithreading, deterministic=False. (PYSR_BASE pop=500/niter=200/maxsize=30
is ~50x oversized for a ~10-node shear; light config gives same verdict in min.)

## 3. Raw results (judge-certified)
| Dataset | MSE | d/d(zbar) | JUDGE verdict | Expected |
|---------|-----|-----------|---------------|----------|
| D0 holo  | 5.44e-06 | 0    | HOLOMORPHIC      | holo OK |
| D1 shear | 1.59e-05 | !=0  | ANTI-HOLOMORPHIC | anti OK |
--report -> PASS (discrimination holo vs anti, both MSE < 1e-3).

Best equations (light config):
  holo : exp(eml(exp(x0-c1),c2)*(c3*sin(x0)))*x0   (holo ops only; no emlstar/my_conj)
  shear: x0 + my_imag(x0*sin((x0*c4)*eml(x0,c5)))
PySR captured the shear's anti-holomorphy via my_imag, NOT emlstar/my_conj.
Since my_imag(z)=(z-conj(z))/(2i) contains a conjugation, the exact judge
derived d/d(zbar)!=0. Verdict does not depend on operator name — the judge
differentiates the real formula. (Same lesson as Penning: marker is not judge.)

## 4. Precision caveat
MSE ~1e-5, not ~1e-32 (Penning rotations). h=-log(1-z) is deeper; light config
recovers an APPROXIMATION, not the exact closed form. For calibration this is
enough: the verdict (holo vs anti) is the result, not the MSE decimals.
